# Kaiba42 Tap

## How do I install these formulae?

`brew install kaiba42/tap/<formula>`

Or `brew tap kaiba42/tap` and then `brew install <formula>`.

Or, in a `brew bundle` `Brewfile`:

```ruby
tap "kaiba42/tap"
brew "<formula>"
```

## Documentation

`brew help`, `man brew` or check [Homebrew's documentation](https://docs.brew.sh).
